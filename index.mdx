与她的海大时光笺 (HNU-TimeLetter) is an interactive visual storytelling website that brings together galgame-style character illustrations and real Hainan University campus locations. Users open a ceremonial envelope to enter an immersive experience — browsing an interactive campus map on desktop or a stamp-album story feed on mobile — uncovering "decisive moments" and the stories behind them.

## How it works

<Steps>
  <Step title="Configure Feishu CMS">
    Set up your Feishu Bitable app credentials and Aliyun OSS bucket in `.env.local`.
  </Step>
  <Step title="Sync data">
    Run `npm run sync` to pull stories, locations, and community creations from Feishu into local JSON files.
  </Step>
  <Step title="Start the dev server">
    Run `npm run dev` and open `http://localhost:3000` to explore the envelope-opening ceremony and interactive campus map.
  </Step>
  <Step title="Deploy to Vercel">
    Push to your `main` branch. Set the Vercel build command to `npm run publish` so it syncs from Feishu and builds in one step.
  </Step>
</Steps>

<Note>
  The project requires a live Feishu Bitable connection — there is no bundled mock data. Make sure your credentials are configured before running `npm run sync`.
</Note>
